# Lunarvim

Lunarvim is a terminal-first AI editor foundation that blends:
- T3-style developer ergonomics
- AstroNvim/Neovim modular architecture
- AI-oriented editing workflows

## Foundation structure

```text
.
├── init.lua
├── lua/
│   └── lunarvim/
│       ├── ai/
│       ├── core/
│       ├── plugins/
│       ├── ui/
│       └── utils/
├── docs/
├── scripts/
└── tests/
```
